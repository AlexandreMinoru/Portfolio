Habilidades

Línguagens de programação e ferramentas que domino

HTML

HTML é a linguagem fundamental para criação de páginas web. Define a estrutura e o conteúdo básico, como textos, imagens e links, utilizando uma série de elementos e tags.

CSS

CSS é uma linguagem usada para estilizar elementos HTML em páginas web, controlando layout, cores e estilos visuais.

JavaScript

JavaScript é uma linguagem de programação para adicionar interatividade a páginas web, manipulando elementos, respondendo a eventos e realizando requisições.

React

React é uma biblioteca JavaScript para construir interfaces de usuário, desenvolvida pelo Facebook.

Java

Java é uma linguagem de programação amplamente utilizada, conhecida por sua portabilidade e versatilidade, sendo utilizada em uma variedade de aplicações, desde desenvolvimento web até aplicativos móveis e sistemas empresariais.

Python

Python é uma linguagem de programação de alto nível, conhecida pela sua simplicidade e legibilidade. É amplamente utilizada em desenvolvimento web, ciência de dados, automação de tarefas, entre outros, devido à sua vasta biblioteca padrão e comunidade ativa.

Figma

Figma é uma ferramenta de design colaborativo em nuvem amplamente usada para criação e prototipagem de interfaces de usuário e designs gráficos.

MySQL

MySQL é um sistema de gerenciamento de banco de dados relacional amplamente adotado por sua eficiência e robustez. É utilizado para armazenar e gerenciar dados de maneira organizada e segura em aplicações web, empresariais e diversas outras soluções tecnológicas.

SmartPark

Uma tentativa de melhorar o trânsito

Desenvolvemos uma aplicação móvel que permite aos usuários ver e reservar vagas de estacionamento em tempo real. Utilizando sensores de ocupação, o app mostra a disponibilidade de vagas próximas e permite reservas antecipadas. A aplicação também oferece navegação guiada até a vaga reservada e pagamento integrado, simplificando a experiência do usuário. A SmartPark foi extremamente elogiada pelos docentes e indicada ao programa Empreenda da instituição.
